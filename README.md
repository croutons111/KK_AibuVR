# KK_AibuVR

**Koikatsu VR**（`KoikatuVR.exe`）の H シーンで、ハンド愛撫アイテムの切り替えと乳首立ち補正を追加する BepInEx プラグインです。

> **β 版** — HF Patch 3.36 + Meta Quest 2（Oculus Link / Air Link）で動作確認済み

---

## 機能

### ハンドアイテム切り替え

通常の VR モードでは胸愛撫に使えるアイテムが固定されています。本プラグインを導入すると、VR コントローラーのボタンまたはキーボードで**その場でアイテムを切り替え**られるようになります。

- **左右独立切り替え** — 左右のコントローラーはそれぞれ別のボタンで独立して切り替えられます
- **ゾーン別自動スキップ** — 現在の愛撫ゾーンで使えないアイテムは自動的に飛ばして次へ進みます
- 愛撫モード・挿入・手・口など、複数の H モードで動作します

**使えるアイテムとゾーンの組み合わせ:**

| アイテム | 胸 | 股間・アナル | 尻 |
|---|---|---|---|
| もみ | ✔ | ✔ | ✔ |
| なでる系 | ✔ | ✔ | ✖（ボディにめり込むためスキップ） |
| 電マ | ✔ | ✔ | ✔ |
| バイブ | ✔（胸のみ） | ✖ | ✖ |

> バイブは胸以外のゾーンでは表示されません（ゲーム仕様）。

---

### 乳首立ち補正

VR の胸愛撫中、ゲームの物理演算によって乳首が胸に埋まってしまうことがあります。本プラグインはこれを補正し、愛撫中も乳首が自然に立った状態を維持します。

- 胸への指愛撫を検出したとき自動で補正を開始し、離れると自動で解除
- 補正量は設定で調整可能（デフォルト: `1.0` = 補正なし）

## インストール

1. [Releases](../../releases) から最新の `KK_AibuVR.dll` をダウンロード
2. `BepInEx/plugins/` フォルダに配置
3. `KoikatuVR.exe` を起動

## 設定

ゲーム内 **BepInEx Configuration Manager**（デフォルト: `F1`）から変更できます。

### Hand Caress（ハンド愛撫）

| 設定項目 | デフォルト | 説明 |
|---|---|---|
| Enabled | `true` | プラグイン全体の有効/無効。`false` でアイテム切り替えと乳首補正が両方無効になります |
| Nipple Protrusion | `1.0` | 胸愛撫中の乳首突出量。`1.0` = 補正なし、値を上げると前方に突き出します（上限 `8.0`） |

### Controls（操作）

| 設定項目 | デフォルト | 説明 |
|---|---|---|
| Cycle Key (Keyboard) | `Tab` | **両手同時**にアイテムを切り替えるキーボードキー |
| Cycle Button - Left Controller | `Grip` | **左手**のアイテムを切り替える VR コントローラーボタン |
| Cycle Button - Right Controller | `Grip` | **右手**のアイテムを切り替える VR コントローラーボタン |

**ボタンの選択肢:** `None`（無効）/ `Grip` / `Menu` / `Touchpad` / `Touchpad_Up` / `Touchpad_Down` / `Touchpad_Left` / `Touchpad_Right`

`None` に設定するとそのコントローラーでの切り替えが無効になります。もう一方の手には影響しません。
